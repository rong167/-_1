**3.停車顯示器**
=
```
語言:QNN+Python+YOLO
開發板:pynq-z2
作品介紹:為了減少駕駛員於停車場找車位的困擾，目前許多停車場已架設停車感應顯示燈，然而當停車格中有物品或是人站在裡面，感應器仍會判斷車格中有車，而停
車顯示燈則會顯示紅燈，因而造成駕駛員錯誤的判斷，因此此作品加入了類神經網路，系統將會判斷停車格中的物品是否為車輛，若是，燈才會開啟(代表停車格中有車)
```

**5.路口來車顯示燈**
=
```
語言:C、C++
開發板:Arduino
作品介紹:近年交通事故眾多，許多意外都發生於巷口追撞事故，因此想透過顯示燈讓行駛於路上的車提前得知巷口有無來車，進而減少事故發生率
```
<div align="center">
  <figure>
  <img src="https://github.com/user-attachments/assets/f1216ade-b5ef-4f08-952a-3452dff2cdcc" width="500" height="300">

  讓駕駛員從遠處就能得知巷口有來車，而蜂鳴器會發出警告提醒駕駛員
